# Lazy CSVParser by Radnaev Tumen

Lazy CVSParser header-only implementation for parsing CSV files.

## Details

Parsing "gogogo.csv" file and write result to the console.


### Requirements


```
CMake VERSION 3.12.1
```

### Build-up

Just write in terminal

```
cmake /lab04
```


### Tests


```
./lab04 -test
```
